03 · Bildverträglichkeit

Schrift über Bildern

Loras mittlerer Strichkontrast bleibt auch auf Fotos lesbar. Ein Königsblau-Schleier über dem Bild sichert den Kontrast zusätzlich ab.

Mitgliedschaft

Wo aus Begegnung Geschäft wird.

Headline in Lora, Vorspann in Source Sans 3, beides auf einem abgedunkelten Bild. Die Lesbarkeit bleibt erhalten.

Faustregel: über Fotos immer einen Königsblau-Schleier mit etwa 60 bis 80 Prozent Deckkraft legen, dann Text in Champagner-Crème setzen.

05 · Entwicklung

Schrift-Tokens

Einbindung über Google Fonts, Zuordnung über CSS-Variablen.

/* <head>: Google Fonts (SIL Open Font License, kostenlos) */
<link href="https://fonts.googleapis.com/css2?family=Lora:wght@400;500;600;700&family=Source+Sans+3:wght@300;400;600;700&display=swap" rel="stylesheet">

:root {
  --serif: /* Überschriften, Wortmarke */ 'Lora', Georgia, serif;
  --sans:  /* Fließtext, UI, Print-Details */ 'Source Sans 3', 'Segoe UI', sans-serif;
}

body        { font-family: var(--sans); font-weight: 400; }
h1, h2, h3  { font-family: var(--serif); font-weight: 600; }
.lead       { font-family: var(--sans); font-weight: 300; }
.caption    { font-family: var(--sans); opacity: .65; }
.ui-label   { font-family: var(--sans); font-weight: 600; text-transform: uppercase; letter-spacing: .2em; }

06 · Alternativen

Wenn eine andere Richtung gewünscht ist

Gleichwertige Wege, falls der Auftritt editorialer oder konsequent moderner ausfallen soll.

Editorialer
Newsreader + Libre Franklin

Etwas mehr Zeitungs- und Journal-Charakter. Newsreader wirkt redaktioneller als Lora, Libre Franklin gibt dem Lauftext mehr Gravität. Geeignet, wenn der Club stark über Inhalte und Vorträge kommuniziert.

Konsequent modern
Source Sans 3 (oder Work Sans) allein

Ein reines Grotesk-System über alle Ebenen. Maximal bildverträglich und zeitgemäß, betont die innovationsgetriebene Seite. Verliert allerdings einen Teil der heraldischen Würde, die die Farben aufbauen.

Empfehlen. Wachsen. Verbinden.

Dieses Schriftmuster rendert die empfohlenen Schriften live und hält zugleich die Farb- und Hintergrundregeln der Markenfibel ein.
