03 · Entwicklung

Design Tokens

Diese CSS-Variablen sind die einzige Wahrheits­quelle für Farbwerte. Niemals Hex-Werte fest verdrahten.

/* Kontor Business Club · Brand Tokens */
:root {
  /* Markenfarben */
  --koenigsblau: #1A2F6E;  /* primär */
  --smaragd:    #15543F;  /* sekundär */
  --champagner: #EFE5C9;  /* erhobene Flächen */
  --gold:       #BD9D5E;  /* Akzent, nie als Textfarbe auf Hell */

  /* System & Funktion */
  --kontorblau: #3457A7;  /* UI-Hierarchie */
  --tinte:      #14181F;  /* Fließtext */
  --pergament:  #F4EEDD;  /* Seitenhintergrund */
}

/* Grundanwendung */
body {
  background: var(--pergament);
  color:      var(--tinte);
}
h1, h2, h3    { color: var(--koenigsblau); }
a             { color: var(--koenigsblau); border-bottom: 1px solid var(--gold); }
.card         { background: var(--champagner); }
.hero, .footer{ background: var(--koenigsblau); color: var(--champagner); }

06 · Typografie

Textfarben-Hierarchie

Tinte für Fließtext, Königsblau für Überschriften. Marken­farben verlieren ihre Wirkung, sobald sie auch im Lesetext erscheinen.

Beispiel auf hellem Grund

Verbunden. Verbindlich. Gewinnbringend.

Dies ist Fließtext in Tinte auf Pergament-Hintergrund. Eingestreute verlinkte Begriffe erscheinen in Königsblau mit feiner Gold­unterstreichung. Die Sekundär­farbe Smaragd bleibt seltenen Auszeichnungen vorbehalten.

Mikrotypografie und Bild­unterschriften treten leicht zurück und stehen in einem abgedimpften Tinten­wert.

Beispiel auf Königsblau

Auf dunkler Fläche tritt Crème an die Stelle von Tinte.

Auf Königsblau-Sektionen wird Fließtext in Champagner-Crème gesetzt. Kicker und feine Akzente in Messinggold.

09 · Validierung

Validierungs-Checkliste

Wenn die folgenden zehn Punkte erfüllt sind, ist die Marken­anwendung im Web korrekt umgesetzt.

  • Seiten­hintergrund   ist Pergament #F4EEDD, niemals reines Weiß.
  • Karten und Panels   sind Champagner-Crème #EFE5C9.
  • Hero, Footer, Statement-Bänder   sind Königsblau #1A2F6E.
  • Fließtext   ist Tinte #14181F, nicht Königsblau.
  • Überschriften   stehen in Königsblau #1A2F6E.
  • Links   sind Königsblau mit feiner Gold­unterstreichung.
  • Gold   erscheint nur als feine Linie, kleines Ornament oder Akzent — niemals als Fließtext­farbe auf hellem Grund.
  • Smaragd   bleibt sparsam für Tags und sekundäre Auszeichnungen.
  • Kontorblau   ist UI-Werkzeug (Hover, Diagramm­kategorien), nicht Lesetext.
  • Reines Weiß   nur für Bilder­karten und technische Daten­flächen verwenden.

10 · Transparenz

Nicht im Scope dieser Fassung

Was bewusst offen geblieben ist und in einer Folge­fassung ergänzt werden sollte.

Offene Punkte Logo-Reinzeichnung, verbindliche Marken­schriften (in dieser Datei sind Marcellus und Spectral angewendet, aber noch nicht offiziell festgelegt), Bildsprache, Iconografie sowie englische und mobile Sonderfälle. Diese Bausteine bauen auf dem hier definierten Farbsystem auf und sollten dieses nicht überschreiben.
